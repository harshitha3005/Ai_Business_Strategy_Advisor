# 🤖 AI Business Strategy Advisor

AI Business Strategy Advisor is a full-stack AI-powered platform that converts a simple business idea into a comprehensive business strategy report using multiple AI agents, Machine Learning, NLP, and LLMs.

## 🚀 Features

* Market Research & Trend Analysis
* Competitor Intelligence
* Customer Sentiment Analysis
* Social Media Trend Detection
* Product Gap Identification
* Pricing Strategy Recommendation
* AI-Generated Business Strategy Reports

## 🛠️ Tech Stack

**Frontend:** React, Vite, Tailwind CSS, Axios

**Backend:** FastAPI, PostgreSQL (Supabase), JWT Authentication

**AI/ML:** Scikit-Learn, BERT, BERTopic, XGBoost, Prophet

**LLM & RAG:** LangChain, OpenAI, FAISS

## 📂 Project Structure

```text
frontend/      # React application
backend/       # FastAPI server
ai_agents/     # AI & ML agents
data/          # Datasets
vector_db/     # FAISS indexes
```

## ⚙️ Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
